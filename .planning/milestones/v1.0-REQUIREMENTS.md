## Original Requirements (Phase 10–11)

### Phase 10: Truth-Grounded Interactive Retrieval

**TCR 1–8 (Truth Contract Requirements):**

1. **Strict grounding** — Answers derivable only from stored atoms (direct/trivial; no inference)
2. **Explicit refusal** — When unsupported, return "I cannot answer based on available knowledge."
3. **Sequential citations** — Use format `[A001]`, `[A002]` per response
4. **Mandatory retrieval** — All responses must go through V3Retriever; no bypass
5. **Indexing delay fallback** — If atoms not yet indexed, fallback to refusal (not LLM completion)
6. **No hard filtering** — Return all relevant atoms regardless of confidence score
7. **Contradictions preserved** — Do not hide, dedupe, or merge conflicting evidence
8. **Validation** — All responses validated for traceability, refusal correctness, no bypass, completeness

**Outcome:** ✅ All satisfied. 98–99% test coverage; 10/10 integration tests pass.

### Phase 11: Synthesis Truth Contract

**Locked Decisions:**

1. V3Retriever ONLY
2. Binary refusal for insufficient evidence
3. Sequential citation format `[A###]`
4. Remove word count minimum (no "MINIMUM 1000 WORDS")
5. Store `atom_ids_used` for regeneration
6. `mission_id` is canonical (all queries/storage filter by mission)
7. Contradictions explicitly stated in prose
8. LLM-structured reports allowed (organizational only)
9. Report = pure transformation (zero inference)

**Outcome:** Initially FAIL (7 blockers). After Phase 11.1 remediation: ✅ All mandatory decisions satisfied.
