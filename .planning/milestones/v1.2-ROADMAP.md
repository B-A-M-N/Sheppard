## Key Decisions

- **Skip-on-failure**: All derivation rules return `None` on error; pipeline never halts.
- **Deterministic claim IDs**: `sha256(rule:sorted_atom_ids:version)[:16]` ensures stability.
- **Ephemeral derived claims**: Never persisted; only live on EvidencePacket.
- **LLM-free derivation**: Pure math and metadata inspection only.
- **Pass 5 stub**: LongformVerifier integration placeholder for future pass; verifier ready for plug-in.
- **No changes to v1 truth contract**: All modifications build atop existing foundation.
